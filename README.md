Movie App

This is a simple movie and TV show application built with React and Vite.
The app uses the TMDB API to display movies, TV series, details, cast, and similar content.

This project was created as a learning project to practice React, routing,
API integration, and component-based development.

Live Demo
https://saint-stream-react.vercel.app/

Tech Stack

- React
- Vite
- React Router
- Axios
- TMDB API

Features

- Browse movies and TV shows
- View details and overview
- Display cast list
- Show similar movies or series

Environment Variables
Create a .env file in the root directory and add:

VITE_TMDB_API_KEY=your_api_key_here

Make sure the .env file is not committed to GitHub.

Run the Project

1. Install dependencies
   npm install

2. Start the development server
   npm run dev
